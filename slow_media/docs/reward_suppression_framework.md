# Reward Suppression Framework
## Preventing Dopamine Exploitation by Design

AIC-Creative does not compete in the attention economy.
We explicitly reject variable reward exploitation.

---

## 1. Prohibited Mechanisms

The following are constitutionally banned:

- Infinite scroll.
- Variable-ratio reward loops.
- Streak mechanics.
- Leaderboards.
- Time-limited artificial urgency.
- Intermittent unpredictable reinforcement.
- Slot-machine timing feedback.

These systems are neurologically manipulative.

---

## 2. The Dopamine Risk Model

We identify risk in three dimensions:

1. Reward Variability
2. Unpredictability Frequency
3. Compulsion Loop Tightness

High values in these dimensions increase addictive potential.

If any module crosses threshold:
- It must be redesigned.
- It cannot be shipped.

---

## 3. Replacement Principles

Instead of reward loops, we implement:

### 3.1 Completion-Based Closure
Experiences end cleanly.

### 3.2 Reflective Feedback
Instead of points:
- Prompt reflection.
- Encourage integration.
- Offer depth.

### 3.3 Non-Competitive Recognition
Acknowledgment without ranking.

---

## 4. Friction as Safeguard

We allow:

- "Would you like to continue?"
- "You have been here for 40 minutes."
- Optional cool-down prompts.

We do not allow:
- "You’re about to lose your streak!"
- Artificial scarcity triggers.

---

## 5. Recommendation Governance

Recommendation systems must:

- Explain why content appears.
- Avoid hyper-personalized compulsion loops.
- Avoid escalating intensity patterns.

No algorithm may optimize for retention alone.

---

## 6. Internal Audit

Every new feature must answer:

- Does this increase compulsive return behavior?
- Does this simulate gambling psychology?
- Does this create dependency loops?

If yes → redesign required.

---

We do not suppress joy.
We suppress exploitation.

We do not remove reward.
We remove coercion.
